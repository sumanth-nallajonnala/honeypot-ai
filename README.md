## Agentic Honey-Pot for Scam Detection

This project implements Problem Statement 01 from the India AI Hackathon
powered by HCL and GUVI.

### Features
- Scam detection
- Autonomous agent replies
- Multi-turn conversation handling
- Intelligence extraction
- Mandatory GUVI callback
- API key authentication

### Endpoint
POST /api/honeypot

### Authentication
x-api-key header is required.
